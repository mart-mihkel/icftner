# In-Context Fine-Tuning for NER with Language Models

## Development

Use [uv](https://docs.astral.sh/uv/) for project management and [typst](https://typst.app/) for typestting.

Run `make install` to setup a local venv, see [Makefile](./Makefile) for other development commands.

## Usage

```bash
uv run cli.py --help
```
